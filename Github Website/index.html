Rival Robotics — Shuttlecock Extraction System
Aug 2025 Onshape

Goal

Make a system to extract shuttlecocks from a tube.

Outcome

  • 3 DOF system able to extract at any angle
  • Easy implementation with the rest of the robot
  • Fully 3D printed and easily assembleable
  • Master Sketch can easily change major values and auto-rebuild

Technical

Uses a 3 DOF system powered by 2 servos for exact rotation, plus 1 motor to spin the roller bars. Uses an embossed hex shape through a bearing connecting to an insert into the tube to transmit power without running a shaft or external systems.

Rival Robotics — Omnidirectional Chassis
Aug 2025 Onshape

Goal

Make a rigid and sturdy chassis using omnidirectional wheels to control the robot's movement.

Outcome

  • Easily machinable aluminum plates
  • Sheet metal parts to reduce complexity and hardware

Technical

Dead axle wheel system for smooth rotation on bearings. Wheels powered via 4 brushless motors with belts. 3D printing for extra parts to reduce machining complexity.

Rival Robotics — Arm System
Jul 2025 Onshape

Goal

Create an arm system with complete freedom of motion to place objects in specific locations easily.

Outcome

  • Differential gearbox for X and Z-axis rotation
  • Virtual 4-bar so input stays in place regardless of arm orientation
  • QDD motors with 3 Nm torque
  • Weight-reduced plates to lower CG for vertically extending system

Technical

Single shaft with dead and live axles at one pivot so the differential can rotate while a stationary pulley rotates in X but is unaffected by Z rotation.

FRC — Cone & Cube Intake

Outcome
